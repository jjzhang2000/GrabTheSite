# GUI界面使用说明

## 启动GUI界面

要启动GUI界面，只需运行以下命令：

```bash
python gui_app.py
```

## GUI界面布局

GUI界面包含以下几个主要部分：

1. **基本配置**：位于顶部，包含目标URL输入框
2. **选项卡区域**：包含以下选项卡
   - **高级配置**：包含抓取深度、最大文件数、输出目录等高级设置
   - **插件配置**：包含插件启用/禁用选项
   - **日志**：显示抓取过程中的日志信息
3. **底部按钮**：包含开始、停止和退出按钮

## 使用步骤

1. **输入目标URL**：在顶部的URL输入框中输入要抓取的网站地址
2. **配置抓取参数**：在高级配置选项卡中设置抓取深度、最大文件数等参数
3. **选择插件**：在插件配置选项卡中选择要使用的插件
4. **点击开始抓取**：点击底部的"开始抓取"按钮开始抓取过程
5. **查看日志**：在日志选项卡中查看抓取过程的详细信息
6. **停止抓取**：如果需要，点击"停止"按钮停止抓取过程
7. **退出程序**：点击"退出"按钮退出GUI界面

## GUI界面特点

- **直观易用**：图形界面比命令行更直观，适合不熟悉命令行的用户
- **实时反馈**：通过日志选项卡可以实时查看抓取过程的状态
- **模块化设计**：不同功能集中在不同的选项卡中，便于管理
- **国际化支持**：支持英文和中文界面
- **配置验证**：在启动抓取前会验证配置的有效性

通过GUI界面，用户可以更方便地使用GrabTheSite工具，无需记忆复杂的命令行参数。
