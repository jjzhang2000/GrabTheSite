# GUI界面使用说明

## 启动GUI界面

要启动GUI界面，只需运行以下命令：

```bash
python grab_gui.py
```

## GUI界面布局

GUI界面包含以下几个主要部分：

1. **基本配置**：位于顶部，包含目标URL输入框
2. **高级配置**：包含抓取深度、最大文件数、输出目录、线程数、延迟设置等
3. **日志**：显示抓取过程中的实时日志信息
4. **底部按钮**：包含开始、停止和退出按钮

## 使用步骤

1. **输入目标URL**：在顶部的URL输入框中输入要抓取的网站地址
2. **配置抓取参数**：在高级配置区域设置抓取深度、最大文件数、线程数等参数
3. **点击开始抓取**：点击底部的"开始抓取"按钮开始抓取过程
4. **查看日志**：在日志区域查看抓取过程的实时输出
5. **停止抓取**：如果需要，点击"停止"按钮停止抓取过程
6. **退出程序**：点击"退出"按钮退出GUI界面

## GUI界面特点

- **直观易用**：图形界面比命令行更直观，适合不熟悉命令行的用户
- **实时反馈**：通过日志区域可以实时查看抓取过程的状态和进度
- **配置持久化**：配置更改会自动保存到配置文件
- **国际化支持**：支持英文和中文界面
- **线程安全**：抓取在后台线程中运行，界面保持响应

通过GUI界面，用户可以更方便地使用GrabTheSite工具，无需记忆复杂的命令行参数。
