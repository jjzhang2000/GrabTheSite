# PDF GUI 使用说明

PDF GUI 是 GrabTheSite 的图形化 PDF 生成工具，提供直观的界面来配置和生成 PDF 文件。

## 启动方式

```bash
python pdf_gui.py
```

## 界面说明

### 基本配置

- **目标 URL**：要抓取的网站地址（必需）
- **抓取深度**：抓取链接的层级深度（默认：1）
- **最大文件数**：最多抓取的页面数量（默认：5）
- **输出目录**：PDF 文件的保存位置

### PDF 配置

- **PDF 文件名**：生成的 PDF 文件名（默认：site.pdf）
- **页面格式**：PDF 页面尺寸
  - A4（默认）
  - Letter
  - Legal
  - Tabloid
- **页边距**：PDF 页面边距，单位毫米（默认：20mm）

### 高级配置

- **请求延迟(秒)**：每次请求之间的间隔时间（默认：0.5秒）
- **无随机延迟**：禁用随机延迟功能
- **线程数**：并行抓取的线程数量（默认：5）
- **用户代理**：HTTP 请求使用的 User-Agent 字符串
- **语言**：界面语言（中文/英文）
- **强制下载所有文件**：忽略时间戳，强制重新下载所有页面
- **不要下载 URL**：排除不需要下载的 URL 列表，支持通配符(*)

### 日志区域

显示抓取和 PDF 生成的实时日志信息，包括：
- 抓取进度
- 下载状态
- PDF 生成状态
- 错误信息

### 操作按钮

- **抓取到 PDF**：开始抓取网站并生成 PDF
- **停止**：停止当前的抓取任务
- **退出**：关闭程序

## 使用步骤

1. **输入目标 URL**
   - 在"目标 URL"字段输入要抓取的网站地址
   - 例如：`https://www.example.com`

2. **配置抓取参数**
   - 设置抓取深度（建议：1-3）
   - 设置最大文件数（建议：5-50）
   - 选择输出目录

3. **配置 PDF 参数**
   - 设置 PDF 文件名
   - 选择页面格式
   - 调整页边距

4. **开始抓取**
   - 点击"抓取到 PDF"按钮
   - 等待抓取和 PDF 生成完成

5. **查看结果**
   - 生成的 PDF 文件保存在指定的输出目录
   - PDF 包含自动生成的书签（目录）

## 书签说明

生成的 PDF 包含智能书签：
- **目录**：使用 📁 图标，空目录自动合并显示（如 `a/.../d`）
- **页面**：使用 📄 图标，点击跳转到对应页面
- **层级**：书签支持层级结构，方便导航

## 语言切换

1. 在"高级配置"区域找到"语言"下拉框
2. 选择"中文"或"英文"
3. 界面文本会自动切换

## 注意事项

- 首次使用需要安装浏览器（Edge 或 Chrome）
- 大网站生成 PDF 可能需要较长时间
- 抓取过程中可以随时点击"停止"按钮中断
- 外部链接在 PDF 中可点击，但需要网络连接
- 建议合理设置"最大文件数"，避免生成过大的 PDF 文件

## 故障排除

### 无法启动浏览器
- 确保系统已安装 Microsoft Edge 或 Google Chrome
- 或运行 `playwright install` 安装 Playwright 内置浏览器

### PDF 生成失败
- 检查输出目录是否有写入权限
- 检查磁盘空间是否充足
- 查看日志区域的错误信息

### 页面抓取不完整
- 增加"请求延迟"，避免请求过快被服务器拒绝
- 检查"不要下载 URL"列表，确保没有误排除
- 启用"强制下载所有文件"选项
