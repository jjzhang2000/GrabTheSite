# Changelog

所有项目的显著变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.0] - 2025-02-20

### 新增

- **核心抓取功能**
  - 智能链接处理，自动转换已下载页面的链接为本地路径
  - 多线程并行抓取，提高抓取速度
  - 增量抓取，基于时间戳只下载更新的内容
  - 断点续传，在抓取中断后可以继续之前的抓取

- **JavaScript渲染**
  - 使用 Playwright 支持动态加载的内容
  - 自动检测系统浏览器
  - 可配置渲染超时时间

- **站点地图生成**
  - 生成 HTML 格式的站点地图
  - 正确反映页面层次结构（树形结构）
  - 使用页面标题作为链接文本

- **排除列表**
  - 支持指定不需要下载的 URL 列表
  - 支持通配符模式匹配

- **国际化支持**
  - 支持英文和中文界面
  - 使用 Python 标准库 gettext 实现翻译功能
  - 可通过配置文件或命令行参数设置语言

- **插件系统**
  - 支持自定义插件扩展功能
  - 提供插件开发 API
  - 保存功能已重构为插件形式

- **GUI界面**
  - 使用 tkinter 开发的图形界面
  - 支持基本配置、高级配置和日志查看
  - 支持实时日志显示
  - 支持国际化

- **配置管理**
  - 使用 YAML 格式的配置文件
  - 支持默认配置和用户自定义配置
  - 配置优先级：命令行参数 > 用户配置 > 默认配置

- **日志系统**
  - 支持控制台日志和文件日志
  - 详细的日志输出
  - 集成到 GUI 界面

- **高级错误处理**
  - HTTP 请求失败自动重试
  - 支持指数退避算法
  - 网络超时的智能处理

- **命令行支持**
  - 丰富的命令行参数
  - 支持帮助信息和参数验证

### 技术细节

- Python 3.x 支持
- 使用 BeautifulSoup 进行 HTML 解析
- 使用 Playwright 进行 JavaScript 渲染
- 使用 tkinter 开发 GUI 界面
- 使用 YAML 进行配置管理
